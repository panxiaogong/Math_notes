## 🌟 内容亮点

- 💎 **线性代数** - 向量、矩阵、特征值和那些令人头疼的变换
- 🔮 **离散数学** - 集合论、图论、逻辑与证明方法
- 🧩 **概率与统计** - 从基础概率到机器学习必备统计知识
- 🚀 **数值计算** - 计算机如何处理数值计算问题
